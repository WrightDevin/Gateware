Detailed Description

A single threaded input library.

The single thread input library is used for high speed game input. You can use this library to get any mouse or keyboard input.

virtual unsigned int GW::SYSTEM::GInput::GetKeyMask ( )
pure virtual

Get the key mask.

The key mask lets the input object know which of the functions below are active by manipulating individual bits of an unsigned int. Values for G_MASK can be found in GKeyDefines.

Return values
G_MASK(_SHIFT, _CONTROL, _CAPS_LOCK, _NUM_LOCK, _SCROLL_LOCK).
