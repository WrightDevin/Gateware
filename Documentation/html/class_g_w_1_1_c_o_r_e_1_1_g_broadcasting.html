Detailed Description

The GBroadcasting Interface is capable of registering & deregistering GListener interfaces. The Broadcaster must then notifiy any listeners of advertised events via the "GListener::OnEvent" callback During registration a GBroadcasting implmenetation may choose to support filtration via bitflags (optional)
